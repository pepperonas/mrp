# App Icon Design Prompt für MRP (Prompt-Optimierer)

## App-Kontext

**App-Name:** MRP (Prompt-Optimierer)  
**Vollständiger Name:** Meta Prompt Refiner / Prompt-Optimierer  
**Plattformen:** Windows, macOS, Linux (Electron Desktop App)  
**Zweck:** KI-gestützte Prompt-Optimierung mit Clipboard-Integration

## App-Funktionalität

MRP ist ein Desktop-Tool, das:
- **Prompts optimiert** mithilfe von KI (OpenAI, Claude, Grok, Gemini)
- **Metaprompts** als Vorlagen verwendet, um normale Prompts zu verbessern
- **Clipboard-Integration** bietet: Prompt kopieren → Shortcut drücken → Optimiertes Ergebnis zurück
- **System Tray Integration** für Hintergrund-Betrieb
- **Global Shortcut** (Ctrl+Shift+O) für schnelle Optimierung

**Kernkonzept:** Ein Tool, das Texte/Prompts transformiert und verbessert - wie ein "Refiner" oder "Optimizer" für Prompts.

## Design-System & Farben

### Primärfarben
- **Brand-Farbe:** `#4A90E2` (Helles Blau) - Hauptfarbe für primäre Aktionen
- **Akzent-Farbe:** `#FF6B35` (Orange) - Für Highlights und Akzente
- **Hintergrund-Primär:** `#1A1C27` (Sehr dunkles Blau-Grau)
- **Hintergrund-Sekundär:** `#2C2E3B` (Dunkles Blau-Grau)

### Design-Philosophie
- **Modern & Minimalistisch:** Klare, einfache Formen
- **Dunkles Theme:** Die App verwendet ein dunkles Farbschema
- **Professionell:** Für Entwickler und Content-Creator
- **Tech-orientiert:** KI/ML-Tool, sollte technisch aber zugänglich wirken

## Icon-Anforderungen

### Visuelle Konzepte

Das Icon sollte folgende Konzepte visuell kommunizieren:

1. **Transformation/Optimierung:**
   - Pfeile, die Transformation symbolisieren (→, ↻, ⬆)
   - Zwei Zustände: Vorher/Nachher, Input/Output
   - Verbesserung/Upgrade-Symbolik

2. **Text/Prompt-bezogen:**
   - Text-Symbolik (Buchstaben, Textzeilen, Dokumente)
   - Prompt-Symbolik (Chat-Bubbles, Textfelder)
   - Schreibwerkzeug-Symbolik

3. **KI/Tech:**
   - Subtile KI-Symbolik (Neural Network, Nodes, Verbindungen)
   - Moderne Tech-Ästhetik
   - Aber: Nicht zu technisch oder abstrakt

4. **Refinement/Polish:**
   - Schleifen/Polieren-Symbolik
   - Upgrade/Verbesserung-Pfeile
   - Optimierung-Symbolik

### Stil-Richtlinien

- **Stil:** Modern, minimalistisch, leicht abgerundet
- **Komplexität:** Mittel - nicht zu einfach, nicht zu komplex
- **Lesbarkeit:** Muss auch in kleinen Größen (16x16px) erkennbar sein
- **Plattform-agnostisch:** Funktioniert auf Windows, macOS und Linux
- **Farbverwendung:** Primär Brand-Blau (#4A90E2), optional Akzent-Orange (#FF6B35) als Highlight
- **Hintergrund:** Kann transparent sein oder dunklen Hintergrund haben (passend zum App-Theme)

### Konkrete Icon-Ideen

#### Option 1: Text-Transformation
- Zwei Textzeilen oder Dokumente
- Pfeil zwischen ihnen (→ oder ↻)
- Eine Zeile "unvollkommen" (gestrichelt, kleiner), eine "perfekt" (klar, größer)
- Brand-Blau als Hauptfarbe

#### Option 2: Prompt-Bubble mit Upgrade
- Chat-Bubble oder Textfeld
- Pfeil nach oben oder Upgrade-Symbol darüber
- Optional: Glow-Effekt (passend zum App-Design)
- Brand-Blau mit subtiler Glow

#### Option 3: Refinement-Symbol
- Schleife oder Zahnrad-Symbol
- Text-Symbolik integriert (Buchstaben oder Linien)
- Transformation-Pfeil
- Modern, minimalistisch

#### Option 4: M/R/P Monogramm
- Stilisierte Buchstaben "M", "R", "P"
- Transformation-Element integriert (Pfeil, Upgrade-Symbol)
- Brand-Blau mit Akzent-Orange
- Modern, minimalistisch

#### Option 5: Text + Optimierung
- Text-Dokument oder Textzeilen
- Optimierungs-Symbol darüber (Pfeil nach oben, Stern, Checkmark)
- Brand-Blau als Hauptfarbe
- Subtile Tech-Ästhetik

### Technische Spezifikationen

#### Größen (für verschiedene Plattformen)
- **macOS:** 16x16, 32x32, 128x128, 256x256, 512x512, 1024x1024 (ICNS Format)
- **Windows:** 16x16, 32x32, 48x48, 256x256 (ICO Format)
- **Linux:** 16x16, 32x32, 48x48, 64x64, 128x128, 256x256, 512x512 (PNG)

#### Format-Anforderungen
- **Vektor-basiert:** SVG als Basis (für Skalierung)
- **Raster-Export:** PNG in verschiedenen Größen
- **Transparenz:** Alpha-Kanal unterstützt
- **Hintergrund:** Kann transparent oder dunkel sein

#### Design-Guidelines
- **Padding:** Mindestens 10% Padding um das Icon-Element
- **Kontrast:** Gute Lesbarkeit auf hellem und dunklem Hintergrund
- **Details:** Nicht zu fein - muss in kleinen Größen funktionieren
- **Farben:** Maximal 2-3 Farben (Brand-Blau, optional Akzent-Orange, optional Weiß/Schwarz)

## Zusätzliche Hinweise

- Das Icon sollte **nicht** zu generisch sein (kein einfaches Dokument oder Text-Symbol)
- Es sollte die **Transformation/Optimierung** betonen
- **KI-Symbolik** kann subtil integriert werden, sollte aber nicht dominieren
- Das Icon sollte **professionell** wirken, aber auch **zugänglich** sein
- **Konsistenz** mit dem dunklen App-Theme ist wichtig
